ChuckWagon

this was my group's final project during our time at the iron yard, called chuckwagon. this is a mobile first web application that allows users to locate nearby food trucks in charleston, sc via geolocation or address search. Here is the blog we kept up with during our time building the ionic app - chuckwagonblog.wordpress.com.

tech: ionic, google maps api, cordova, gulp

links: demo github

Jeopardy

here i created a jeopardy game using angular js and a jeopardy question bank api. it is set up with a landing page view with six categories, that randamize when you refresh the page to start a new game. each category contains points that calculate to keep score.

tech: html5, css3, angular js, jservice api

links: demo github

Skate

personal project i created for people struggling with vocalizing their feelings. basic premise of the app is to type however you're feeling in that moment and pop your feelings away. i built the back-end using ruby on rails and used materialize, paperjs and plain jquery for the front-end of the site.

tech: html5, css3, sass

links: demo github
